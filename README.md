**[Infinity Water Bucket](https://www.curseforge.com/minecraft/mc-mods/infinity-water-bucket)** is a **Minecraft mod** that allows the Infinity enchantment for Water Buckets using an anvil, providing an infinite portable water source. Also, options for infinite Lava, Milk, and Powdered Snow buckets.
